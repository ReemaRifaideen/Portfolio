# Portfolio

## 📖 About the Project
This repository contains a simple personal portfolio website created using only HTML. The project is designed to present personal information, skills, projects, and contact details through a clean and structured web page.

The purpose of this project is to practice HTML fundamentals and understand the structure of basic web development.

## 🚀 Technology Used
- HTML5

## 🛠️ Tools
- VS Code

## ✨ Features
- Simple and clean design
- Personal introduction section
- Skills section
- Project showcase section
- Contact information section
- Beginner-friendly structure

## 📁 Project Structure

```text
Portfolio/
│── index.html
│── README.md
```

## ⚙️ Installation & Usage

### Clone the repository

```bash
git clone https://github.com/ReemaRifaideen/Portfolio.git
```

### Navigate to the project folder

```bash
cd Portfolio
```

### Run the project

Open the `index.html` file in your web browser.

This project helps in understanding:

- Creating webpage sections
- Organizing content effectively
- Building a simple portfolio layout
